# Kreeda Prerana Scouts

Kreeda Prerana Scouts is an Android application developed using Kotlin and Jetpack Compose for athlete scouting and performance tracking. The app is designed to help coaches, trainers, and sports organizations manage athlete information, monitor performance, and simplify the scouting process.

## Features

- Athlete profile management
- Performance tracking and analysis
- Modern UI built with Jetpack Compose
- Easy data organization for scouts and coaches
- Fast and responsive Android application

## Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio
- Gradle

## Project Structure

- `app/` – Main application source code
- `gradle/` – Gradle configuration files
- `build.gradle.kts` – Build configuration
- `settings.gradle.kts` – Project settings

## Purpose

The main goal of this project is to create a simple and efficient scouting platform for identifying and tracking athlete performance. It helps in maintaining organized records and improving sports talent evaluation.
